# 이감 모의고사 주문 취합 시스템 (잇올 독학관리학원)

잇올 독학관리학원(수만휘) 40개 지점의 이감 모의고사 주문을 학생 개별 주문부터 학원 승인, 본사 취합까지 한 흐름으로 처리하는 시스템.

주문 흐름: 학생 주문(student.html) → 학원 취합·승인(branch.html) → 본사 취합·확정(admin.html) → 이감 전달

| 파일 | 사용자 | 용도 |
|---|---|---|
| student.html | 학생 | 지점별 주문링크(?b=주문코드)로 접속하는 학생 개별 주문 페이지 |
| branch.html | 지점 담당자 | 지점별 관리링크(?k=관리코드)로 자기 지점 주문 취합·승인/반려 |
| index.html | 지점 담당자 | 지점 일괄 주문 페이지 (보조) |
| admin.html | 본사 관리자 | 취합 관리 (주문 취합·리포트·지점/상품 관리·수동 입력) |
| config.js | 공통 | Apps Script API URL 설정 |
| apps-script/code.gs | 백엔드 | 구글 시트 연동 API (Apps Script에 붙여넣기) |
| 설치가이드.md | 본사 관리자 | 최초 1회 설치 및 운영 안내 |

설치·운영 방법은 [설치가이드.md](설치가이드.md) 참고.
